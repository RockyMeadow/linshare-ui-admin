Structure de courriel

Cette vue vous permet de créer des modèles de strucutre de courriel. Il vous faudra créer un modèle HTML & Text. Un modèle de structure vous permet d'organiser la disposition des différentes parties (salutation, message, logo, footer) au sein d'un courriel.

  • ${mailSubject} Titre / Sujet du courriel
  • ${image} Image, logo, entête du courriel
  • ${personalMessage} Message personnel qui est ajouté au message de base ${body}.
  • ${greetings} Message ---
  • ${body} Correspond au contenu que vous saisissez dans la vue 'contenu de courriel'
  • ${footer} Correspond au contenu que vous saisissez dans la vue 'Pied de courriel'

Dans la vue 'Configuration de courriel' vous pourrez créer des modèles de courriel en choisissant pour :

  1. Structure de courriel HTML / Text le template créé ici en version HTML & Text
  2. Pied de courriel le pied de courriel qui sera ajouté à la variable ${footer}
  3. Contenu de courriel les contenus de courriel (en version EN et FR) qui seront ajoutés à la variable ${body}
