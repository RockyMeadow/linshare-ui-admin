MIME policies

In this view you can filter the uploaded files e.g. forbid upload a videos (MPEG, etc.).

Operation :

  • Click on the « + » button and give a name to your MIME type filter
  • Choose the filter that you added than a list of available MIME types is displayed;
  • Enable or disable the MIME type you want than save.
  • You can than select this filter in'Manage domains' menu when you create or edit a domain
