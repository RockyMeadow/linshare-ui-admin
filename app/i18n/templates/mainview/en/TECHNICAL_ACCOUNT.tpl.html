Technical accounts

This view allows you to create new accounts with specific delegation rights.

Permissions details:

  • 'Delegation' : allows to make operations for other users via the delegation API. To do that you should select the permissions that you give to this account. (rights). By default any permission is given to it.
    For more indication you can see : http://download.linshare.org
  • 'Upload proposition' : Upload proposition is an external module. So it requires a special account to get access to LinShare and recover its configuration
