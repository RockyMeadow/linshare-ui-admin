Модели домена

После создайте подключения к каталогу LDAP, необходимо создать модель домена.

Этот шаблон позволит вам определить соответствие между полями имени, фамилии и электронной почты {{'COMMON.NAME' | translate}} и атрибуты вашего каталога. {{ 'COMMON.NAME' | translate }} требует 4 разных запросов для правильной работы :

  • запрос LQL (Ldap Query Language) идентификации : запросить найти пользователя в каталоге на этапе аутентификации из введенного им идентификатора (по электронной почте)
  • запрос LQL (Ldap Query Language) пользователь : запрос, чтобы найти пользователя в каталоге при поиске консультаций, редактирования, совместного использования, ...
  • запрос LQL (Ldap Query Language) автозаполнение по всем атрибутам : permet l'autocomplétion d'un utilisateur à partir du motif saisi
  • запрос LQL (Ldap Query Language) d'autocomplétion sur le nom et prénom : позволяет автозавершение пользователя из введенного шаблона
