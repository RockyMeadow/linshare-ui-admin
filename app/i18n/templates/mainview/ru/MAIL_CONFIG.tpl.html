Конфигурация почты

Это окно позволяет создавать структуры электронной почты для ваших доменов. Из информации, введенной в представлениях :

  • 'Структура электронной почты' позволяет создавать шаблоны структуры электронной почты, различные макеты
  • 'Нижний колонтитул электронной почты' для компоновки ножки электронной почты
  • 'Содержимое электронной почты' управлять содержимым уведомлений, отправляемых автоматически пользователям приложения

После настройки различных шаблонов структуры электронной почты вы сможете в «Конфигурация домена»' (при создании / редактировании одного из ваших доменов), выбрать конфигурацию электронной почты.
