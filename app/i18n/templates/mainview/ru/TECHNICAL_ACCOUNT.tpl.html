Технические аккаунты

Это окно позволяет создавать учетные записи с определенными правами делегирования.

Сведения о правах :

  • 'делегирование' : позволяет выполнять операции для других пользователей через API-интерфейс делегирования. Чтобы выполнить эти операции, вы должны выбрать список разрешений (прав) , которые вы хотите предоставить этой учетной записи. По умолчанию не выбраны разрешения.
    Для определения того, как работает API-интерфейс делегирования, вы можете обратиться к своей документации, доступной по адресу : http://download.linshare.org
  • 'Загрузить предложение' : Загружайте предложение, являющееся внешним модулем приложения, для подключения к LinShare требуется специальная учетная запись и восстановление ее конфигурации. Эта учетная запись посвящена этому использованию.
