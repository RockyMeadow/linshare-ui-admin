Персонализируйте приветствие для пользователей, подключенных к приложению «LinShare».

Вы можете создать несколько приветствий по доменам и связать их с разными поддоменами.
Чтобы создать приветствие, вам просто нужно дублировать существующее приветствие, чтобы сделать это:

  • В дереве левых доменов вы найдете список доменов: родители (только для чтения) и список ваших доменов и поддоменов (в режиме записи) .
    Вы можете дублировать приветствие с помощью кнопки «duplicate» в списке приветствий или путем предварительного просмотра приветствия. После повторного запроса откроется окно, вам нужно будет ввести имя и домен, в который вы хотите скопировать приветствие.
  • С помощью кнопки «добавить» откроется окно с просьбой выбрать домен, чтобы выбрать шаблон приветствия, вам нужно будет выбрать новое имя и копию, это будет выполняться в текущем домене.

После этого создания вы сможете изменить приветствие и назначить его тем доменам по вашему выбору, которые смогут извлечь выгоду из него, то есть во время , вы сможете выбрать там приветствие, предлагаемым списком будут приветствия, назначенные для домена. По умолчанию, если приветствие не предоставляется, список будет предлагать только приветствие по умолчанию.
