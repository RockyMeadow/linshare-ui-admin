Это поле представляет собой базу поиска для всех пользователей в вашем каталоге LDAP, например «ou = People, dc = linshare, dc = org. Это поле будет введено как переменная с именем «домен» во всех запросах шаблона домена для каждого поиска ldap.
Если пользователь принадлежит нескольким доменам, вы должны указать соответствующее связанное с ним имя домена в поля поиска, чтобы процесс аутентификации точно идентифицировал его.
